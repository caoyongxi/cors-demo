## CORS 跨域demo 验证

## 运行项目

首先为了方便测试，配置一下你的host指向:  
    你本机的ip地址  huoyun-test.djtest.cn  
    你本机的ip地址  test.djtest.cn

```bash
$ git clone https://github.com/caoyongxi/cors-demo.git
$ cd cors-demo
$ npm install
$ npm run server 
$ npm run client

```

## 调试server端代码
